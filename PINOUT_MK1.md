# Mk1 Pinout Reference

This guide summarizes the Mk1 harness pin assignments for the Replica dashboards. It is intended as a quick-reference wiring aid; always confirm signal types and expected voltages before making permanent connections.

## Mk1 connector mapping

| Pin | Signal / description |
| --- | --- |
| 1 | Illumination and low-beam supply. |
| 2 | MASSE 31 ground reference. |
| 3 | TANK fuel-level sender. |
| 4 | TEMP coolant temperature sender. |
| 5 | KL 1 tachometer signal. |
| 6 | UHR permanent +12 V. |
| 7 | KL 56 high-beam signal. |
| 8 | OIL (HIGH) 1.8 bar pressure switch. |
| 9 | OIL (LOW) 0.3 bar pressure switch. |
| 10 | Diesel glow indicator. |
| 11 | CHOKE input (unused). |
| 12 | KL 61 generator lamp. |
| 13 | Blinker input (combined left/right). |
| 14 | KL 15 ignition supply. |
